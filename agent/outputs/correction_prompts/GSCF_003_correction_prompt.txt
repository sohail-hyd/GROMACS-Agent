5. Post-processing and MSD analysis commands
5.1 Remove periodic boundary condition jumps

For diffusion analysis, molecules must be made whole and continuous across periodic boundaries.

gmx trjconv -s md.tpr -f md.xtc -o md_nojump.xtc -pbc nojump

When asked for group, select:

System

Or use:

echo "System" | gmx trjconv -s md.tpr -f md.xtc -o md_nojump.xtc -pbc nojump
5.2 Create an index group for water oxygen atoms

For water self-diffusion, it is common to calculate MSD using water oxygen atoms.

gmx make_ndx -f md.gro -o index.ndx

Inside the interactive menu, type:

a OW
name 3 Water_Oxygen
q

If your water oxygen atom name is different, check it using:

head water_box.gro

Common names are:

OW
OH2
O
5.3 Calculate MSD and diffusion coefficient

Use gmx msd:

gmx msd -s md.tpr -f md_nojump.xtc -n index.ndx -o msd_water.xvg -beginfit 1000 -endfit 9000

When asked to select group, choose:

Water_Oxygen

Or use a non-interactive command if the group name is recognized:

echo "Water_Oxygen" | gmx msd -s md.tpr -f md_nojump.xtc -n index.ndx -o msd_water.xvg -beginfit 1000 -endfit 9000

Important: do not use invalid options such as:

-type no

The diffusion coefficient will be printed in the terminal and also written in the output comments of msd_water.xvg.

5.4 View the MSD curve
xmgrace msd_water.xvg

Or save the data for plotting elsewhere:

cp msd_water.xvg water_msd_300K.xvg

The MSD curve should become approximately linear after the initial ballistic and short-time relaxation region.

This file contains MSD versus time.

The terminal output from gmx msd should report something like:

D[Water_Oxygen] = X.XXXX (+/- Y.YYYY) 1e-5 cm^2/s

For liquid water at 300 K, a reasonable experimental self-diffusion coefficient is around:

2.3 × 10^-9 m^2/s

GROMACS may report diffusion in:

10^-5 cm^2/s

Conversion:

1 × 10^-5 cm^2/s = 1 × 10^-9 m^2/s

So:

2.3 × 10^-5 cm^2/s = 2.3 × 10^-9 m^2/s

The exact result depends strongly on water model, box size, trajectory length, thermostat/barostat, and fitting interval.

Possible causes:

Pressure or density is not equilibrated
PBC correction was not applied

Fix:

Check temperature and density:
