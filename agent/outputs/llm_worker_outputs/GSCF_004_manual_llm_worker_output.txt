GROMACS Workflow: Hydrogen-Bond Number of Pure Water at 300 K
1. Required input files

For the post-processing hydrogen-bond analysis, the minimum required files are:

md.tpr
md.xtc
topol.top

Recommended full workflow input/output files:

water_box.gro
topol.top
minim.mdp
nvt.mdp
npt.mdp
md.mdp
em.gro
nvt.gro
npt.gro
md.gro
md.xtc

2. Topology preparation steps

Create a pure water simulation box.

Step 1: Create empty topology file
touch topol.top

Edit topol.top:

#include "oplsaa.ff/forcefield.itp"
#include "oplsaa.ff/spce.itp"

[ system ]
Pure water at 300 K

[ molecules ]
Step 2: Generate solvated water box
gmx solvate -cs spc216.gro -o water_box.gro -box 3.0 3.0 3.0 -p topol.top

This adds water molecules and updates topol.top automatically.

Expected files:

water_box.gro
topol.top

4. GROMACS command sequence
Step 1: Energy minimization
gmx grompp -f minim.mdp -c water_box.gro -p topol.top -o em.tpr
gmx mdrun -deffnm em

Expected files:

em.gro
em.edr
em.log
em.tpr
Step 2: NVT equilibration
gmx grompp -f nvt.mdp -c em.gro -r em.gro -p topol.top -o nvt.tpr
gmx mdrun -deffnm nvt

Expected files:

nvt.gro
nvt.xtc
nvt.edr
nvt.log
nvt.tpr
Step 3: NPT equilibration
gmx grompp -f npt.mdp -c nvt.gro -r nvt.gro -t nvt.cpt -p topol.top -o npt.tpr
gmx mdrun -deffnm npt

Expected files:

npt.gro
npt.xtc
npt.edr
npt.log
npt.cpt
npt.tpr
Step 4: Production MD
gmx grompp -f md.mdp -c npt.gro -t npt.cpt -p topol.top -o md.tpr
gmx mdrun -deffnm md

Expected files:

md.tpr
md.xtc
md.gro

5. Hydrogen-bond post-processing analysis
Main command
gmx hbond -f md.xtc -s md.tpr -num hbond_water.xvg

GROMACS will ask for two interactive selections:

Select a group for donors:
Select a group for acceptors:

For pure water, choose the water group for both selections.

Common group names may be:

Water
SOL
System

Recommended selection:

Water
Water

or, if the group is named SOL:

SOL
SOL

This means the analysis calculates hydrogen bonds between water molecules.

Non-interactive version

If the group is called Water, use:

printf "Water\nWater\n" | gmx hbond -f md.xtc -s md.tpr -num hbond_water.xvg

If the group is called SOL, use:

printf "SOL\nSOL\n" | gmx hbond -f md.xtc -s md.tpr -num hbond_water.xvg

If GROMACS requires group numbers instead of names, first check the available groups:

gmx make_ndx -f md.tpr -o index.ndx

Then quit by typing:

q

After checking the correct water group number, for example group 4, run:

printf "4\n4\n" | gmx hbond -f md.xtc -s md.tpr -n index.ndx -num hbond_water.xvg
6. Expected output files

Main hydrogen-bond output:

hbond_water.xvg

Depending on GROMACS version, the output may also be named or equivalent to:

hbnum.xvg

The file contains the hydrogen-bond number time series.

Typical columns are:

time    number_of_hydrogen_bonds

You can inspect the file using:

head hbond_water.xvg

To remove comment/header lines:

grep -v "^[@#]" hbond_water.xvg > hbond_water_clean.dat

To calculate average hydrogen-bond number:

awk '!/^[@#]/ {sum+=$2; n++} END {print "Average H-bond number =", sum/n}' hbond_water.xvg
7. Optional visualization

Use Grace:

xmgrace hbond_water.xvg

Or convert to plain data:

grep -v "^[@#]" hbond_water.xvg > hbond_water.dat

Then plot time versus hydrogen-bond number.

8. Common possible errors and fixes
Error 1: Missing trajectory file

Error example:

File input/output error: md.xtc does not exist

Fix:

ls

Check whether the production run finished and produced:

md.xtc
md.tpr

If not, rerun:

gmx mdrun -deffnm md
Error 2: Missing TPR file

Error example:

Required option -s not provided or md.tpr not found

Fix:

gmx grompp -f md.mdp -c npt.gro -t npt.cpt -p topol.top -o md.tpr
Error 3: Wrong donor and acceptor groups

Problem:

Hydrogen-bond number is zero or physically unreasonable

Cause:

You may have selected a non-water group or selected only oxygen atoms without proper donor hydrogen information.

Fix:

For pure water, select water as both donor and acceptor group:

Water
Water

or:

SOL
SOL
Error 4: No water group appears

Fix:

Create an index file:

gmx make_ndx -f md.tpr -o index.ndx

Then use it:

gmx hbond -f md.xtc -s md.tpr -n index.ndx -num hbond_water.xvg
Error 5: Short trajectory gives unstable average

Problem:

A very short trajectory, for example 10–100 ps, may produce a hydrogen-bond time series but the average is not reliable.

Fix:

Use a longer production trajectory, preferably at least:

1 ns for workflow validation
5–10 ns or longer for stronger statistical analysis

If the trajectory is short, report clearly:

The hydrogen-bond result is used only to validate the workflow, not as final publication-quality data.
9. Short validation note

The workflow is valid if:

gmx hbond -f md.xtc -s md.tpr -num hbond_water.xvg

successfully produces:

hbond_water.xvg
