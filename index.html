🧪 GROMACS-Agent

Text-to-Workflow Generation for Molecular Dynamics
School of Energy and Environment, Southeast University

📄 A multi-agent framework for topology-aware GROMACS workflow generation, evaluation, correction, and selected safe execution

  • Text-to-Workflow – Convert natural language tasks into complete GROMACS workflows (files, commands, outputs, validation).
  • Multi-Agent Architecture – Manager, Planner, Retriever, Worker, Evaluator, and Code Executor agents work together.
  • Evaluation & Correction – A dedicated Evaluator agent scores workflows and suggests fixes in a Reflexion-style loop.
  • Curated Datasets – Ships with GSCF (workflow generation) and GEQS (evaluation) datasets.
  • Safe Execution – Selected safe-execution checks for allow-listed commands (RDF, MSD, density, etc.).
📑 Paper: GROMACS-Agent: Text-to-Workflow Generation for Molecular Dynamics (chemRxiv, 2026)
📊 Baseline Improved evaluator score from 6.6/10 → 10/10 across representative tasks.
⚡ Speed Local workflow generation in ~0.05 s (vs. ~47 s manually in a pilot test).

📦 Datasets & Models

  • GSCF – GROMACS Script Construction for Fine-tuning (workflow generation records).
  • GEQS – GROMACS Expert Quality Scoring (evaluation records).
  • Available on Hugging Face (coming soon) and in the repository.

git clone https://github.com/sohail-hyd/GROMACS-Agent.git
