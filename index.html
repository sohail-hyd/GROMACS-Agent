GROMACS‑Agent

Text‑to‑Workflow Generation for Molecular Dynamics
A multi‑agent framework for topology‑aware GROMACS workflow generation, evaluation, correction, and selected safe execution.

📄 chemRxiv 2026 ⭐ GitHub

🏛️ School of Energy and Environment, Southeast University

🔍 Introduction

Molecular dynamics (MD) simulation links atomic structure to thermodynamic, transport, and interfacial properties, but translating a scientific question into an appropriate GROMACS workflow remains a practical barrier.

A complete GROMACS workflow spans structure, topology, MDP, TPR, trajectory, energy, index, and output files — and many errors are scientific rather than syntactic, surviving execution while quietly invalidating the result.

🚀 GROMACS‑Agent

A multi‑agent prototype that treats GROMACS automation as text‑to‑workflow generation rather than text‑to‑code generation. Combines Manager, Planner, Retriever, Worker, Evaluator, and Code Executor agents.

🧰 Key Features

📦

Text‑to‑Workflow

Convert natural language tasks into complete GROMACS workflows: files, commands, outputs, and validation.

🧠

Multi‑Agent System

Manager, Planner, Retriever, Worker, Evaluator, and Executor agents collaborate seamlessly.

📚

Curated Datasets

GSCF (workflow generation) and GEQS (evaluation) datasets with 100 records each.

🧑‍🔬

Expert‑Evaluated

Improved evaluator‑assessed correctness from 6.6/10 → 10/10 across representative tasks.

⏱️

Time Efficient

Local workflow generation in ~0.05 s vs. ~47 s manually in pilot tests — a ~99.9% reduction.
